# 🌍 CodeAlpha — Language Translation Tool

A Python desktop application that translates text between 100+ languages using Google Translate.

---

## 📸 Preview

> A clean, modern Claymorphism-styled translator with rounded cards, purple accents, and smooth interactions.

---

## ✨ Features

- 🌐 Translate between **100+ languages**
- 🎨 Modern **Claymorphism UI** built with CustomTkinter
- 📋 **Copy** translation to clipboard with one click
- 🗑️ **Clear** button to reset both text fields

## ⚙️ Installation

**1. Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_LanguageTranslator.git
cd CodeAlpha_LanguageTranslator
```

**2. Create a virtual environment**
```bash
python -m venv env
env\Scripts\activate
```

**3. Install dependencies**
```bash
pip install -r requirements.txt
```

**4. Run the app**
```bash
python translatorV1.5.py
```

## 🚀 How to Use

1. Type or paste any text in the **"Enter text to translate"** box
2. Select your **source language** from the left dropdown (or leave as Auto Detect)
3. Select your **target language** from the right dropdown
4. Click **Translate**
5. View the result in the **Translation** box
6. Click **Copy** to copy the result or **Clear** to reset
